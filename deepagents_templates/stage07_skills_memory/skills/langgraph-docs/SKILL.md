# langgraph-docs

## Overview
该技能用于把 LangGraph 学习目标拆成可执行计划，并提供阶段化建议。

## Instructions
1. 明确用户当前水平（入门/进阶）。
2. 将学习目标拆成周计划 + 每日任务。
3. 每天至少包含“阅读 + 动手 + 复盘”三个动作。
4. 输出时先给结论，再给步骤。
