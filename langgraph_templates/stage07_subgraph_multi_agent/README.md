# Stage 07: Subgraph & Multi-Agent

## 目标
- 学会将子流程封装为独立子图（Subgraph）。
- 理解主图如何通过节点调用子图。
- 体验多 Agent 协作的基本模式。

## 核心概念
- **Subgraph**：一个独立编译的 StateGraph，可以作为节点嵌入到另一个图中。
- **状态映射**：子图可以有自己的 State schema；嵌入主图时会按同名字段自动映射，复杂场景也可手写映射节点。
- **Multi-Agent**：多个"专家"子图各司其职，主图做路由和编排。

## 任务
1. 运行 `main.py`，观察主图如何调度两个子图。
2. 给某个子图增加一个新节点。
3. 尝试添加第三个子图。

## 运行
```bash
uv run main.py
```

## 验收标准
- 主图能根据条件路由到不同子图。
- 子图各自独立运行，结果汇总到主图 state。
